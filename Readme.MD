# BankAccount Lab


This Java application simulates a simple bank account with deposit, withdrawal, and balance check functionalities.


## Files


- `BankAccount.java`: Contains the class definition for a bank account.

- `Main.java`: Contains the main method to test the functionality of the `BankAccount` class.
